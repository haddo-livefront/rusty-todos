# rusty-todos
Hacking with Rust

## How to Run it

Run the following commands at the root of the project:

```sh
# Add a new task
$ cargo run -- add "Buy groceries"

# Add another task
$ cargo run -- add "Learn Rust ownership concept"

# List all tasks
$ cargo run -- list

# Mark the task with index 1 as done

$ cargo run -- done 1
```
